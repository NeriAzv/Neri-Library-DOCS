Introduction to Neri Library
============================

The **Neri Library** is an advanced Python library for task automation, browser interaction, API manipulation, and more. It is designed to facilitate the creation of robust and efficient scripts, especially for web automation and integration with external systems.

Key Features:
- **Driver**: Initialization and configuration of browsers (Chrome, Firefox, Edge, etc.).
- **API**: Integration with external APIs for HTTP requests.
- **System**: System operations such as PDF extraction, directory creation, and message display.
- **Elements**: Manipulation of web page elements using Selenium.
- **CLI**: Command-line interface to simplify task execution.
